1) LIPIETZ, apresenta duas caracteriscas da divisão do trabalho social, a divisão horizontal e a divisão vertical

	divisão horizontal {
		- diz respeito ao espaço fisíco ao qual a divisão do trabalho é feita
		- ex:
			- ramos de atividades
			- cidade-campo
			- divisão inter-urbana
}

	divisão vertial {
		- diz respeito as relações sociais dentro da divisã do trabalho
		- ex: {
			- grupos sociais
			- dominantes e dominados
		}
}
